# Invitation de Mariage - Chayma

Projet complet HTML / CSS / JavaScript.

## Contenu
- Écran d'ouverture type invitation reçue
- Enveloppe animée
- Carte d'invitation élégante
- Localisation intégrée : Walagh, Djerba, Tunisie
- Bouton Google Maps
- Compte à rebours
- Pétales/fleurs animées
- Musique douce générée en JavaScript, sans fichier MP3

## Tester dans VS Code
1. Ouvre le dossier dans VS Code
2. Lance avec Live Server
   ou dans le terminal :
   python -m http.server 5500
3. Ouvre :
   http://localhost:5500

## Modifier
- Texte et dates : index.html
- Style/couleurs : style.css
- Musique douce et compte à rebours : script.js
